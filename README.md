# desafioJavaScript
Desafio Javascript - Detalle y mostrar total

Al presionar en el boton calcular
1. Se toman los valores de los input de los boletos deseados.
2. Se almacenan en una variable con un string concatenado que mostrara el detalle de la cantidad de boletos deseados y el mismo es aplicado a un div vacio que muestra los detalles.
3. Se hace el calculo de la cantidad de boletos por el precio correspondiente y se suman sus valores para mostrar el total a pagar.
